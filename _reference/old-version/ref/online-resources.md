## Search Tips

### WebFetch Strategy
1. Start with the specific help page URL from above
2. If the topic has sub-pages, follow links from the fetched content
3. For API specifics, use the REST API Help URLs

### WebSearch Queries
- `site:therefore.net <topic>` — Search within Therefore's domain
- `"Therefore" DMS <topic>` — General web search
- `"Therefore" API <endpoint-name>` — Search for specific endpoint info

## URL Pattern for Help Pages
```
https://www.therefore.net/help/{year}/en-us/AR/{page}.html
```
- Replace `{year}` with `2025` (latest)
- Replace `{page}` with the topic identifier (e.g., `ar_categories`, `integration-api`)
- For German: replace `en-us` with `de-de`
