# Referencia - TheJTool Old Version

Esta carpeta contiene la **versión anterior** de TheJTool como referencia para implementar funcionalidades faltantes en el proyecto actual.

## Qué revisar en la versión anterior

### ✓ Generación de XML
- Cómo se generaban los XML completos con colores
- Manejo de paleta de colores en la generación
- Estructura de categorías, secciones y campos

### ✓ Campos tipo Tabla
- Cómo se definían las columnas de una tabla
- Estructura de datos para tablas anidadas
- Renderizado en el editor

### ✓ Preview
- Cómo se mostraba el preview en tiempo real
- Validación de estructura
- Visualización de campos y pestañas

### ✓ Otros detalles
- Patrones de validación
- Manejo de estado
- Estructura de componentes

## Cómo usar

1. Abre `old-version/src` para revisar los componentes
2. Compara con la versión actual en `src/`
3. Usa como referencia para:
   - Entender flujos de datos
   - Ver cómo se manejaban características específicas
   - Copiar lógica probada
